# 内联文本语义

这些元素必须嵌套在```<p>```元素里。

## 段落

<p>我是段落。</p>
``` html
<p>我是段落。</p>
```



## 粗体

<p>我是<b>粗体。</b></p>
<p>我是<strong>粗体</strong>，还有语气<strong>重</strong>的意思。</p>
``` html
<p>我是<b>粗体。</b></p>
<p>我是<strong>粗体</strong>，还有语气<strong>重</strong>的意思。</p>
```



## 斜体

<p>我是<i>斜体</i>。</p>
<p>我是<em>斜体</em>，还有语气<em>重</em>的意思。</p>
``` html
<p>我是<i>斜体</i>。</p>
<p>我是<em>斜体</em>，还有语气<em>重</em>的意思。</p>
```



## 删除字

<p>我是<del>删除字</del>。</p>
``` html
<p>我是<del>删除字</del>。</p>
```



## 插入字

<p>我是<ins>插入字</ins>。</p>
``` html
<p>我是<ins>插入字</ins>。</p>
```



## 小号字

<p>我是<small>小号字</small>。</p>
``` html
<p>我是<small>小号字</small>。</p>
```



## 高亮

<p>我是<mark>高亮</mark>。</p>
``` html
<p>我是<mark>高亮</mark>。</p>
```



## 上下标
<p>这是<sup>上</sup>中<sub>下</sub>。</p>
``` html
<p>这是<sup>上</sup>中<sub>下</sub>。</p>
```
